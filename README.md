# Cobra CLI Template

Steps followed:

```bash
brew install go
go mod init github.com/zhouselena/resilienceScore
go get -u github.com/spf13/cobra@latest
```

Run without build:

```bash
go run main.go
```

Run with build:

```bash
bash run-cobra-cli-template.sh
./bin/cobra-cli-template command1
...
```
